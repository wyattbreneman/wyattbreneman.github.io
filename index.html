Project Description

This project analyzes bus stop distribution and transit accessibility across Orange County, California. Using GIS spatial analysis, the goal is to evaluate how effectively public transit infrastructure serves communities and identifies gaps in coverage.

Project Goals

  • Map bus stop and route accessibility across Orange County
  • Analyze spatial distribution of transit services
  • Identify underserved areas with limited transit access
  • Build an interactive GIS web dashboard for exploration

Functionality

  • Interactive filtering by bus route
  • Dynamic indicators for bus stop counts per route
  • Clickable map features with route details
  • Embedded ArcGIS Dashboard visualization

Results & Findings

The analysis reveals spatial differences in transit coverage across Orange County, showing that some routes have higher concentrations of bus stops while others serve more dispersed or underserved regions. These patterns highlight opportunities for improving transit equity and accessibility planning.
